2. Краткая теория

Метод опорных векторов (SVM) ищет разделяющую гиперплоскость с максимальным зазором между классами. Точки, лежащие на границах зазора, называются опорными векторами.
Жесткий зазор (hard margin) требует линейной разделимости и отсутствия ошибок внутри полосы. Мягкий зазор (soft margin) допускает нарушения, регулируется параметром C: большие C — жесткий зазор, малые C — более широкая полоса с возможными ошибками.
Ядра (kernel trick) позволяют строить нелинейные разделяющие поверхности без явного перехода в пространство высокой размерности. Основные ядра:

linear — линейная граница.
rbf — радиально-базисная функция, гибкая нелинейная граница.
poly — полиномиальное ядро.
Для регрессии используется SVR (Support Vector Regression), где ищется функция, отклоняющаяся от истинных значений не более чем на заданную величину ε.
SVM вводит параметр C, который разрешает некоторым точкам нарушать границу зазора (попадать внутрь или даже на «чужую» сторону).

SVM использует трюк с ядром: неявно переносит данные в пространство большей размерности, где они становятся линейно разделимыми, и там строит гиперплоскость

4. Решение самостоятельных заданий

Влияние параметра C
При больших C (например, 1000) зазор стремится к жёсткому, модель пытается безошибочно разделить обучающие данные, что может привести к переобучению на зашумлённых данных. При малых C (0.1) зазор расширяется, допускаются ошибки, модель становится более обобщающей.
Предсказание для точки внутри зазора
Если новая точка попадает внутрь полосы зазора, метод decision_function возвращает значение между -1 и 1 (а не по модулю >1, как для уверенных областей). Класс определяется по знаку: положительное → класс 1, отрицательное → класс 0. Таким образом, даже внутри зазора модель даёт определённое предсказание.
Сравнение ядер на данных make_circles

linear: accuracy низкая (~0.5), граница — прямая, не способна разделить кольца.
poly (degree=3): accuracy ~0.9, граница нелинейная, но может быть избыточно сложной.
rbf: наилучший результат, плавная граница.
Вывод: для нелинейно разделимых данных необходимо использовать нелинейные ядра, RBF часто является оптимальным выбором по умолчанию.

Время обучения в зависимости от размера выборки
SVM имеет квадратичную или кубическую сложность от числа объектов. При количестве точек более 10 000 обучение с RBF-ядром занимает десятки секунд, что делает метод менее пригодным для очень больших наборов данных.
